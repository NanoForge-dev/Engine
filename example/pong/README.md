## setup.sh

Allow you init your game

```shell
./setup.sh
```

## update.sh

Allow you to update your lib

```shell
./update.sh
```

## build.sh

Allow you to build project and copy it to the loader

```shell
./build.sh Loader
```

## run.sh

Allow you to build project and copy it to the loader, and then run the loader (Ctrl + C to stop it)

```shell
./run.sh Loader
```
